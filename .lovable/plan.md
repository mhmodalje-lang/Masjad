

# خطة إصلاح النصوص المقصوصة وتأخير الأذان

## المشاكل المكتشفة

### 1. مشكلة النصوص المقصوصة في الهيدر
كل الصفحات التي تستخدم نمط الهيدر المنحني (`gradient-islamic` + `absolute -bottom-6 rounded-t-[2rem]`) تعاني من نفس المشكلة: النص العلوي والأزرار قريبة جداً من الحافة العلوية ومن المنحنى السفلي. المشكلة الأساسية هي أن `pb-20` أو `pb-14` لا يكفي لإبعاد المحتوى عن المنحنى المغطي.

**الصفحات المتأثرة:**
- `Qibla.tsx` - العنوان وأزرار التحكم مقصوصة
- `Stories.tsx` - العنوان قريب من المنحنى  
- `Tasbeeh.tsx` - العنوان وزر إعادة التعيين
- `PrayerTracker.tsx` - العنوان مقصوص

### 2. مشكلة عامة في كل الصفحات
النصوص والكلمات في بطاقات وعناصر مختلفة قد تكون مقصوصة بسبب عدم وجود `overflow-hidden` أو `break-words` أو `min-w-0` بشكل منتظم.

---

## الحل

### إصلاح الهيدر في كل الصفحات
- زيادة `pt-14` إلى `pt-16` لإبعاد المحتوى عن الحافة العلوية
- إضافة `relative z-10` لكل محتوى الهيدر ليظهر فوق المنحنى
- إضافة `overflow-hidden` على الـ wrapper الرئيسي لكل صفحة

### الملفات والتغييرات:

| الملف | التغيير |
|-------|---------|
| `src/pages/Qibla.tsx` | تغيير `pt-14` → `pt-16`، إضافة `relative z-10` للـ flex container |
| `src/pages/Stories.tsx` | تغيير `pt-14` → `pt-16`، إضافة `relative z-10` |
| `src/pages/Tasbeeh.tsx` | تغيير `pt-14` → `pt-16` (z-10 موجود) |
| `src/pages/PrayerTracker.tsx` | تغيير `pt-14` → `pt-16`، إضافة `relative z-10` للعنوان |
| `src/pages/Duas.tsx` | فحص وتطبيق نفس النمط |
| `src/pages/More.tsx` | فحص وتطبيق نفس النمط |
