

# خطة: إعادة تصميم الصفحة الرئيسية على نمط تطبيق الأذان

## ما سيتم بناؤه

إعادة هيكلة الصفحة الرئيسية بالكامل لتتطابق مع تجربة تطبيق Athan، مع إضافة نظام أهداف يومية تفاعلي وبطاقات مهام قابلة للتحديد.

## الميزات الجديدة

### 1. نظام الأهداف اليومية التفاعلي (Daily Goals System)
بطاقات مهام يومية قابلة للتحديد (check/uncheck) تشمل:
- **صيام رمضان** — عداد يوم الصيام الحالي (اليوم X من رمضان) مع زر تحديد
- **أذكار متعددة** — "قوّي إيمانك (0/7)"، "ثواب ثلث القرآن (0/3)"، "للتوبة (0/11)" — كل واحد مع عداد وتقدم دائري
- **استمع إلى القرآن** — مع وقت متبقي
- **حدّث شخصًا عن الإسلام** — مهمة بسيطة

### 2. الأهداف المقترحة
قسم "الأهداف المقترحة" بأسفل الأهداف اليومية مع زر (+) لإضافتها:
- قراءة سورة الملك قبل النوم
- أداء صلاة نافلة
- إعطاء صدقة
- صيام يوم الاثنين/الخميس
- اقرأ سورة الكهف يوم الجمعة

### 3. بطاقة إدارة الإشعارات
بطاقة مرئية بتصميم جذاب (مثل الصورة) تعرض معاينة إشعار الهاتف مع زر "الذهاب إلى الإعدادات"

### 4. تحسين بطاقة أكمل أهداف اليوم
- إضافة عداد رمضان (0/2 رمضان) 
- شريط تقدم أفقي
- زر "افحص ملفك الشخصي"

## التغييرات التقنية

### قاعدة البيانات
جدول جديد `daily_goals` لحفظ تقدم الأهداف:
```sql
CREATE TABLE daily_goals (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id uuid NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE,
  date text NOT NULL,
  goal_key text NOT NULL,
  completed boolean DEFAULT false,
  progress int DEFAULT 0,
  target int DEFAULT 1,
  UNIQUE(user_id, date, goal_key)
);
ALTER TABLE daily_goals ENABLE ROW LEVEL SECURITY;
```

### الملفات المتأثرة

1. **`src/pages/Index.tsx`** — إعادة هيكلة كاملة:
   - بطاقة الأهداف المحسّنة مع عداد رمضان وشريط تقدم
   - قسم الأهداف اليومية التفاعلية (بطاقات قابلة للتحديد)
   - بطاقة إدارة الإشعارات الجديدة
   - قسم الأهداف المقترحة مع زر إضافة
   - ترتيب الأقسام: Hero → أهداف اليوم → الصلاة التالية + العد التنازلي → شريط رمضان → أوقات الصلاة → الأهداف اليومية التفاعلية → بطاقة الإشعارات → القرآن → دعاء اليوم → الأهداف المقترحة → الوصول السريع → التقويم

2. **`src/components/DailyGoals.tsx`** — مكوّن جديد:
   - قائمة أهداف مع تقدم دائري ملون (أخضر للصلاة، برتقالي للذكر، بنفسجي للقرآن)
   - كل هدف: اسم + وصف + عداد + زر تحديد
   - حفظ في localStorage للضيوف وفي قاعدة البيانات للمسجلين

3. **`src/components/SuggestedGoals.tsx`** — مكوّن جديد:
   - قائمة أهداف مقترحة مع زر (+) لإضافتها للأهداف اليومية
   - تصميم بطاقات بيضاء مستديرة

4. **`src/components/NotificationCard.tsx`** — مكوّن جديد:
   - بطاقة تشرح إدارة الإشعارات مع زر إعدادات
   - قابلة للإغلاق (×)

### تصميم البطاقات
- خلفية بيج فاتحة (`bg-[#F5F0E8]`) للصفحة كاملة
- بطاقات بيضاء مستديرة (`rounded-3xl bg-white`)
- دوائر تقدم ملونة بجانب كل هدف
- أيقونة (⋮) لقائمة خيارات كل هدف
- ألوان: أخضر للصلاة، برتقالي للذكر، بنفسجي للقرآن، أخضر داكن للأزرار
